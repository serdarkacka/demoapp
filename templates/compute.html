Enter an array of elements using yaml format in the first form.
The second form accepts an integer representing start index of the array.
The third form accepts an integer representing end index of the array.
The elements of the array from start to end are concatenated as a result.

An example input for the first form is:
- A
- B
- C
- D

If now the second form has value 1 and the third value has value 3 the result will be BCD.
Note: If start index is less than 0, or end index is greater than or equal to the length of the array the empty string will be returned.
